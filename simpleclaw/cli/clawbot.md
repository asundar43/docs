# `simpleclaw clawbot`

Legacy alias namespace kept for backwards compatibility.

Current supported alias:

- `simpleclaw clawbot qr` (same behavior as [`simpleclaw qr`](/cli/qr))

## Migration

Prefer modern top-level commands directly:

- `simpleclaw clawbot qr` -> `simpleclaw qr`
