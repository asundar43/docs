# `simpleclaw config`

Config helpers: get/set/unset values by path. Run without a subcommand to open
the configure wizard (same as `simpleclaw configure`).

## Examples

```bash
simpleclaw config get browser.executablePath
simpleclaw config set browser.executablePath "/usr/bin/google-chrome"
simpleclaw config set agents.defaults.heartbeat.every "2h"
simpleclaw config set agents.list[0].tools.exec.node "node-id-or-name"
simpleclaw config unset tools.web.search.apiKey
```

## Paths

Paths use dot or bracket notation:

```bash
simpleclaw config get agents.defaults.workspace
simpleclaw config get agents.list[0].id
```

Use the agent list index to target a specific agent:

```bash
simpleclaw config get agents.list
simpleclaw config set agents.list[1].tools.exec.node "node-id-or-name"
```

## Values

Values are parsed as JSON5 when possible; otherwise they are treated as strings.
Use `--strict-json` to require JSON5 parsing. `--json` remains supported as a legacy alias.

```bash
simpleclaw config set agents.defaults.heartbeat.every "0m"
simpleclaw config set gateway.port 19001 --strict-json
simpleclaw config set channels.whatsapp.groups '["*"]' --strict-json
```

Restart the gateway after edits.
