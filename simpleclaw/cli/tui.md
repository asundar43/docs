# `simpleclaw tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
simpleclaw tui
simpleclaw tui --url ws://127.0.0.1:18789 --token <token>
simpleclaw tui --session main --deliver
```
